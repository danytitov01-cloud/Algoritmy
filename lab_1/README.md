# Algoritmy
1. Что называется алгоритмом?

Алгоритм — это точная и конечная последовательность действий, которая приводит к решению задачи.

2. Какие свойства должен иметь алгоритм?

Алгоритм должен быть:
	•	Понятным — шаги однозначны и ясны
	•	Конечным — обязательно заканчивается
	•	Результативным — даёт результат
	•	Дискретным — состоит из отдельных шагов
	•	Массовым — подходит для разных входных данных

3. В чём отличие линейного алгоритма от алгоритма с ветвлением?
	•	Линейный алгоритм — команды выполняются строго по порядку, без условий
	•	Алгоритм с ветвлением — дальнейшие действия зависят от условия (если → то → иначе)

⸻

4. Для чего используются циклы?

Циклы используются для многократного повторения одних и тех же действий без переписывания кода.

5. Какие типы циклов существуют в программировании?

Основные типы циклов:
	•	Цикл с предусловием (while)
	•	Цикл с постусловием (do…while)
	•	Цикл с параметром (for)
